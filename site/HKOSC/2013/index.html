Time: 9am to 6pm

Venue: Run Run Shaw Creative Media Centre, City University of Hong Kong.

Registration / RSVP: http://hkosc2013.eventbrite.com

Web Site: http://opensource.hk/2013

Agenda: http://opensource.hk/2013/agenda
